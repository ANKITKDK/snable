Description
The common theme throughout this course is that we will be designing with users in mind. that being said, How do they do ? How do they act ? those are the questions we will strive to answer in this module as we dig into popular research techniques and methodologies.
Skills You Will Learn
  1. Prototyping, wireframing, user flows, mockups
  2. Visual design and design software
  3. User research and usability testing
  4. Information architecture
Syllabus
Module 1 : User centric design research.
  • Qualitative and Quantitative research practices
  • Information architecture and flow
  • User need Identification
  • User experience Interviewing.
  • Persona creation.
Module 2 : User Interface Development.
  • Atomic design
  • UI Grids and composition
  • Color Theory
  • Heuristic Evaluation
  • Typography
  • Interaction Design and Technography
Module 3 : Visual prototyping and Wireframing.
  • Figma
  • Adobe XD
  • Invision
  • Google Slides
  • Storyboarding
  • Decision mapping
  • Flow Diagrams
  • Prototyping
Duration - 6 Months
